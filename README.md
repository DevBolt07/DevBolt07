<details>
<summary><b>NutriLabel AI — RAG-Enhanced Food Label Analysis</b></summary>

<br/>

An AI-driven food label analysis system that combines OCR, Retrieval-Augmented Generation over Open Food Facts and USDA FoodData Central, and LLM reasoning to deliver clinically grounded ingredient risk scores — personalized for users with diabetes and hypertension. Deployed as a mobile-first application on Firebase Hosting.

| Attribute | Details |
|---|---|
| **Stack** | React · FastAPI · PostgreSQL · Supabase · Pytesseract · RAG · Firebase |
| **Scale** | Mobile-first · Multi-user · Real-time health alerts |
| **Performance** | Benchmarked across latency · accuracy · retrieval precision |
| **Security** | Supabase Auth · Row-Level Security · personalized data isolation |
| **Impact** | Accepted for oral presentation — AdComSys 2026, UEM Kolkata |
| **Repository** | [![Repo](https://img.shields.io/badge/GitHub-label--insight--pro-8B5CF6?style=flat-square&logo=github)](https://github.com/DevBolt07/label-insight-pro) |

This project bridges the gap between consumer food products and medical-grade nutrition intelligence, enabling users to make informed dietary decisions through AI-powered label interpretation backed by two large open health datasets.

</details>
